The Making of HypeAura

Behind the design, colors, and code — how the HypeAura website was created step by step.

Design Mockups

We started by making mockups in Figma to plan out how everything would look. This helped us figure out the layout and spacing before we started coding.

1. Home page

The home page has a big hero image at the top to grab people's attention right away. We wanted visitors to see the brand vibe immediately when they land on the site.

The registration form is on the home page so users can sign up easily without clicking around looking for it. Once they register, their name shows up on other pages which makes it feel more personal.

Colors: We used black for the navbar to make it look sleek and professional. White backgrounds keep everything clean and easy to read. The red button stands out so people notice it and click to register.

2. Men's page

The men's collection page starts with a hero banner that says "Men's Collection" so visitors immediately know what page they're on.

We organized the products in a grid layout because that's how most shopping websites do it. This way customers already know how to use it without getting confused.

Each product card shows the important stuff: product image, name, description, and price. This gives customers all the info they need to decide if they want to buy it.

Button colors: The "Add to Cart" buttons are blue instead of red because they have a different purpose than the register button. Using different colors for different actions helps customers understand what each button does without getting confused.

3. Women's page

The women's page uses the same layout as the men's page to keep everything consistent. We didn't want customers to get confused when they switch between different collection pages.

The hero banner image is different to show the women's style and vibe. This way customers immediately know they're looking at women's products.

Design consistency: We kept the same colors, button styles, and layout for all collection pages. This makes the website look professional and clean, not crazy with different colors everywhere. It's easier for customers to navigate when everything works the same way.

4. New Arrivals page

We needed a 6th page for the project requirements, so we came up with the idea of a "New Arrivals" page to show the latest products that just came in.

This page shows both men's and women's new products mixed together. Customers can see all the newest stuff in one place without having to check multiple pages.

Same layout: We kept the same grid layout as the other collection pages for consistency. Customers already know how to browse products, so there's no need to change how it works.
